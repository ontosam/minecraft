# Ezra's Blocks 🧱

A gentle, friendly block‑building world made for a 6‑year‑old. It looks and feels
like Minecraft, but everything is **simpler, calmer, and easier to control**:

- 🌞 Always daytime — **no monsters, nothing scary, and you can't "die."**
- 🧱 **Build** anything with a big palette of colourful blocks.
- ⛏️ **Dig** to dig and reshape the world.
- 🐷 **Cute animals** wander around — walk up and **pet** one and it becomes your
  pet and follows you, with little hearts! 💗
- 🕹️ **Easy controls** designed for small hands (no keyboard, no menus).
- 💾 Creations **save automatically** on the iPad.
- 📴 Works **offline** once it's been opened the first time. No ads, no purchases,
  no chat, no strangers — it's completely self‑contained.

It runs right in the iPad's web browser (Safari) and can be added to the home
screen so it opens fullscreen and behaves just like a real app.

## 📲 Getting it onto Ezra's iPad

You only have to do this once. The easiest way (free, no software to install):

### Step 1 — Put it online with GitHub Pages
1. Open the project on **github.com** (the `ontosam/minecraft` repository).
2. Click **Settings** (top of the page) → **Pages** (left sidebar).
3. Under **Build and deployment → Source**, choose **“Deploy from a branch.”**
4. For **Branch**, pick **`claude/tender-mayer-550bb0`** and the folder **`/ (root)`**,
   then click **Save**.
5. Wait about a minute and refresh. GitHub will show a link near the top like:

   **`https://ontosam.github.io/minecraft/`**

That link is the game. 🎉

> Prefer it on your main site address? Merge this branch into `main` and pick
> `main` in step 4 instead — the link works the same way.

### Step 2 — Open it on the iPad and add it to the home screen
1. On the iPad, open **Safari** and go to the link from Step 1.
2. Tap the **Share** button (the square with an arrow pointing up).
3. Tap **“Add to Home Screen,”** then **Add**.
4. Close Safari and tap the new **Ezra's Blocks** icon. It opens fullscreen. ✅

Turning the iPad **sideways (landscape)** works best.

*(Alternative with no GitHub steps: on a computer, download the project as a ZIP
from GitHub’s green **Code** button, unzip it, and drag the folder onto
**app.netlify.com/drop** — it gives you an instant link you can open on the iPad.)*

## 🎮 How to play (you can show Ezra)

| What he wants to do | How |
|---|---|
| **Walk** | Hold and drag on the **left** side of the screen — a little circle appears, like a joystick. |
| **Look around** | Drag on the **right** side of the screen (nice and slow). |
| **Build a block** | Point the **+** in the middle at a spot, then tap the green **Build** button. |
| **Dig a block** | Point the **+** at a block, then tap the orange **Dig** button. |
| **Jump** | Tap the blue **Jump** button. (He also *auto‑hops* up small steps so he won't get stuck.) |
| **Pick a colour/block** | Tap one of the squares in the row along the top. |
| **Pet an animal** 🐾 | Walk close to an animal and tap the pink **Pet** button. It hops happily, hearts appear, and it follows him as a pet! |
| **Go back home** 🏠 | Tap the house button if he ever wanders too far. |
| **Brand‑new world** ✨ | Tap the sparkle button (it asks "are you sure?" first). |
| **Sound on/off** 🔊 | Tap the speaker button. |

### For the curious (technical notes)
- Built as a tiny **WebGL** voxel engine with **no external libraries** — so it
  loads instantly, works offline, and runs on essentially any iPad.
- Everything is plain static files (`index.html`, `styles.css`, `js/`), so any
  static web host works.
- App icons are generated by `scripts/make-icons.mjs` (`node scripts/make-icons.mjs`).
